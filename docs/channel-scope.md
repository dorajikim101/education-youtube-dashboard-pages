# channel-scope

## 실제 ingest 대상 채널
현재 자동 수집 대상은 아래 4개 채널만 포함한다.

1. 입시덕후
2. 교육대기자TV
3. EBSi
4. 공부의신 파인만

설정 파일:
- `config/channels.seed.json`

## 운영 원칙
- 과거 전체 영상 backfill 하지 않음
- 각 채널의 최신 영상 1개부터 시작
- 이후 신규 업로드만 자동 수집

## 포함 대상
- 교육 정보
- 입시 조언
- 학생/학부모 정보
- 공부 전략
- 학교/교육 트렌드
- 제도 변화

## 제외 대상
- 교과 개념 설명
- 문제풀이
- 과목 해설
- 강의형 학습 콘텐츠
- 직접적인 수업/커리큘럼형 콘텐츠

## EBSi 예외 규칙
EBSi는 정보성/전략성 영상만 포함하고,
교과 설명/개념 강의/문항 해설 계열은 제외한다.
