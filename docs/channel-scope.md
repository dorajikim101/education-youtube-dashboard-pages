# channel-scope

## 실제 ingest 대상 채널
현재 자동 수집 대상은 아래 5개 채널만 포함한다.

1. 입시덕후
2. 교육대기자TV
3. EBSi
4. 공부의신 파인만
5. 조작가의 스몰빅클래스

설정 파일:
- `config/channels.seed.json`

## 운영 원칙
- 과거 전체 영상 backfill 하지 않음
- 기본은 각 채널 최신 영상 1개부터 시작
- 조작가의 스몰빅클래스는 예외적으로 최신 **실제 공개 영상 2개**부터 시작
- 이후 신규 업로드만 자동 수집

## 포함 대상
- 교육 정보
- 입시 조언
- 학생/학부모 정보
- 공부 전략
- 학교/교육 트렌드
- 제도 변화

## 제외 대상
- 교과 개념 설명
- 문제풀이
- 과목 해설
- 강의형 학습 콘텐츠
- 직접적인 수업/커리큘럼형 콘텐츠
- 예정 영상
- Premiere 대기 영상
- 알림받기만 가능한 미공개 예정 영상

## EBSi 예외 규칙
EBSi는 정보성/전략성 영상만 포함하고,
교과 설명/개념 강의/문항 해설 계열은 제외한다.

## 공개 상태 규칙
우리 ingest 기준은 **실제로 공개 업로드되어 재생 가능한 영상만**이다.
즉, 채널에 보여도 아래 상태는 수집하지 않는다.
- 예정됨
- Premiere 대기
- 알림받기 버튼만 있는 상태
- 아직 공개되지 않은 예약 영상

## 메모
조작가의 스몰빅클래스는 현재 seed에 추가했으며,
최종 채널 handle/url은 실제 수집기 연결 단계에서 확정해야 한다.
